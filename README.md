**Product Management App**

**Features**
User Authentication - Secure login with JWT token management
Product Management - Full CRUD operations for products
Smart Search & Filters - Search by title and filter by category
Modern UI - Responsive design with Tailwind CSS
State Management - Centralized state with Pinia
Route Protection - Automatic redirects based on auth status
Mobile Friendly - Fully responsive across all devices

**Tech Stack**
Frontend Framework: Vue 3 (Composition API)
Routing: Vue Router 4
State Management: Pinia
Styling: Tailwind CSS
HTTP Client: Axios
Build Tool: Vite
API: DummyJSON

**Installation**
Prerequisites:
Node.js 16.0 or higher
npm or yarn

**Quick Start**
Clone the repository
bash
git clone https://github.com/your-username/vue-product-management-app.git
cd vue-product-management-app

**Install dependencies**
bash
npm install
Start development server

bash
npm run dev
Open your browser
Navigate to http://localhost:5173

**Troubleshooting**
**Common Issues**
Login fails with "Invalid credentials"
Ensure you're using the correct demo credentials
Check your internet connection
Products not loading
Verify API endpoint availability
Check browser console for errors

**Build errors**
Clear node_modules and reinstall: rm -rf node_modules && npm install
Ensure Node.js version is 16+
